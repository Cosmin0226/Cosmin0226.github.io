Bootcamp 2020: Vintage Art

My first programming project that was made in Processing is called Vintage Art. When doing this project, I have used loops, trigonometry and a bit of randomness. I have aimed to make it pleasing for the eye, using an ocean-like colour for the moving lines, and a sand-like colour for the background. The way it moves is the same as waves do. This is so looking at it will not be tiring for the human eye.
